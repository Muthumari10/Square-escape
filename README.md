# Square Escape (Gesture-Based Game)

## Overview
Square Escape is a gesture-controlled game developed using computer vision techniques. The game uses real-time hand tracking to let players control gameplay through gestures instead of keyboard or mouse input.

## Features
- Real-time hand tracking using MediaPipe
- Gesture-based game control
- Collision detection
- Score tracking
- Auto-restart functionality
- Smooth interactive gameplay

## Tech Stack
- Python
- OpenCV
- MediaPipe

## How It Works
The system detects hand movements through a webcam using MediaPipe and translates gestures into game controls. Players avoid collisions, track scores, and continue gameplay with automatic restart support.

## Installation
```bash
git clone https: https://github.com/Muthumari10/Square-escape.git
cd square-escape
python main.py
```
